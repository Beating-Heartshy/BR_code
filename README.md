
# BR_code v1.0.0

## 🎉 发布说明

我们很高兴宣布手势识别系统的首个正式版本发布！这个项目是基于我们最新研究论文的实现。

### 🌟 主要特性

- **Aaptive Difference Perception-based Frame Extraction**: 基于感知差异的帧提取
- **Visual Persistence-based Dynamic Trajectory Capture**: 基于视觉持久性的动态轨迹捕捉


### 🔧 技术栈

- Python 3.8+
- TensorFlow 2.13.0
- NumPy 1.24.3
